## RECOMMENDED DEPLOYMENT: VERCEL (FREE & EASIEST)

### Why Vercel?
- âœ… FREE for personal projects
- âœ… Automatic deployments on git push
- âœ… Global CDN (fast worldwide)
- âœ… HTTPS by default
- âœ… Perfect for Vite/React apps
- âœ… Zero configuration needed

### Steps to Deploy:

1. CREATE GITHUB REPOSITORY
   ```bash
   # Initialize git (if not already done)
   git init
   git add .
   git commit -m "Initial commit: eGain Easy Insight"
   
   # Create repo on GitHub.com, then:
   git remote add origin https://github.com/YOUR_USERNAME/egain-easy-insight.git
   git branch -M main
   git push -u origin main
   ```

2. DEPLOY TO VERCEL
   - Go to: https://vercel.com/new
   - Sign in with GitHub
   - Click "Import Project"
   - Select your repository
   - Vercel auto-detects Vite settings:
     * Build Command: npm run build
     * Output Directory: dist
     * Install Command: npm install
   - Click "Deploy"
   
3. GET YOUR URL
   - Deployment completes in ~2 minutes
   - You get: https://YOUR_PROJECT.vercel.app
   - Share this URL with eGain panel

4. FUTURE UPDATES
   - Just push to GitHub
   - Vercel auto-deploys (takes ~1 minute)

## ALTERNATIVE 2: GITHUB PAGES (FREE, MORE MANUAL)

### Setup:
```bash
# Install gh-pages
npm install --save-dev gh-pages

# Add to package.json scripts:
"predeploy": "npm run build",
"deploy": "gh-pages -d dist"

# Update vite.config.js:
export default defineConfig({
  base: '/egain-easy-insight/',  // Your repo name
  plugins: [react()],
  // ... rest of config
})

# Deploy
npm run deploy
```

URL: https://YOUR_USERNAME.github.io/egain-easy-insight

**Note**: GitHub Pages requires the base path config in vite.config.js
